Each folder is a standalone piece you can copy-paste into your setup. No fluff, just stuff that works in production.

## What is this?

I use Claude Code daily with 16+ MCP servers, custom agents, and hooks that keep things from blowing up. This repo is where I open-source the pieces that might save you a `DROP TABLE` in prod.

## Setup

Each drop has its own README with copy-paste instructions. Most hooks go into `~/.claude/settings.json`.
